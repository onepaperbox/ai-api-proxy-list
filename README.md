## 📖 项目介绍

随着 AI 大模型的普及，越来越多的开发者需要调用 Claude、GPT、Gemini 等模型的 API。然而，由于网络限制、支付门槛、价格等因素，许多开发者选择通过 **API 中转/代理服务** 来访问这些模型。

本项目旨在：

- 📋 **全面收录** 市面上的 API 中转/代理服务信息
- 🔍 **客观对比** 各服务的价格、模型支持、特色功能
- 🤝 **社区共建** 欢迎大家补充和纠错
- ⚠️ **仅供参考** 不构成任何推荐或背书

## 🌟 Introduction

As AI large models become popular, more developers need to call APIs like Claude, GPT, and Gemini. However, due to network restrictions, payment barriers, pricing, and other factors, many developers choose to access these models through **API relay/proxy services**.

This project aims to:

- 📋 **Comprehensively collect** API relay/proxy service information on the market
- 🔍 **Objectively compare** pricing, model support, and features
- 🤝 **Community-driven** Contributions and corrections are welcome
- ⚠️ **Reference only** Does not constitute any recommendation or endorsement

## 📐 评估维度

选择 API 中转服务时，建议从以下维度进行评估：

| 维度 | 说明 | 如何判断 |
|:---:|:---|:---|
| 🟢 **可用性** | 服务是否稳定在线，API 调用成功率 | 查看 SLA 承诺、社区反馈 |
| ⚡ **延迟** | 从请求到首个 Token 的时间 (TTFT) | 自行测试或查看官方基准 |
| ❌ **错误率** | API 调用返回错误的比率 | 小规模压测，关注 429/500 |
| 🔄 **并发** | 同时支持的请求数量 | 查看文档限制或实测 |
| 🔍 **保真度** | 返回结果是否与官方 API 一致 | 对比官方 API 输出质量 |
| 💰 **价格** | 相对官方价格的折扣比例 | 综合考量折扣与稳定性 |
| 🛡️ **合规性** | 数据安全与隐私保护 | 查看隐私政策、数据流转 |
| 📞 **售后** | 出现问题时的响应速度 | 测试客服响应时间 |

> 💡 **提示**：建议先用小额度测试多个服务，再决定长期使用哪个。
