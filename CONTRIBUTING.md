# 🤝 贡献指南 Contributing Guide

感谢你对 **AI API Proxy List** 项目的关注！欢迎通过各种方式参与贡献。

---

## 📋 如何添加新的服务商

### 方式一：提交 Pull Request（推荐）

1. **Fork** 本仓库
2. 在你的 Fork 中编辑 `README.md`
3. 在对应的分类表格中添加新的服务商信息
4. 提交 **Pull Request**，标题格式：`[Add] 服务商名称`

### 方式二：提交 Issue

如果你不熟悉 Git 操作，可以提交 Issue，格式如下：

```markdown
### 服务商信息

- **名称**：
- **官网**：
- **类型**：API 中转 / AI 基础设施 / 大模型厂商 / 云厂商 / 账号服务 / 国际聚合 / 公益站点
- **支持模型**：
- **核心特色**：
- **价格参考**：
```

---

## ✏️ 修改已有信息

如果你发现已有信息存在错误或需要更新：

1. 同样通过 PR 或 Issue 提交
2. 请说明修改原因和依据（如官网截图、最新定价页面等）
3. 标题格式：`[Fix] 服务商名称 - 修改内容`

---

## 📝 信息规范

### 添加服务商时，请遵循以下规范：

| 字段 | 要求 | 示例 |
|:---:|:---|:---|
| 名称 | 使用品牌官方名称，可附英文 | 盛算云 Shengsuanyun |
| 官网 | 必须是可访问的官方网址 | `https://shengsuanyun.com` |
| 类型 | 从已有分类中选择 | API 中转 |
| 支持模型 | 列出主要支持的模型 | Claude / GPT / Gemini |
| 核心特色 | 简明扼要，1-2 个亮点 | SLA 99.7%，企业级网关 |
| 价格参考 | 简明价格信息 | 首充 10% off |

### 注意事项

- ✅ 信息必须**真实可验证**，优先引用官方来源
- ✅ 价格信息注明**时效性**，如"截至 2025 年 X 月"
- ✅ 核心特色**客观描述**，避免营销话术
- ❌ 不得添加虚假、误导性信息
- ❌ 不得恶意贬低竞争对手
- ❌ 不接受付费排名或置位请求

## 🏷️ 分类说明

提交时请选择正确的分类：

| 标识 | 分类 | 说明 |
|:---:|:---|:---|
| `vendor` | 🤖 大模型厂商 | 模型原厂提供的官方 API |
| `infra` | 🏗️ AI 基础设施 | 企业级推理网关和基础设施 |
| `cloud` | ☁️ 云厂商 | 云服务提供商的 AI 平台 |
| `relay` | 🔀 API 中转 | 第三方 API 转发服务 |
| `global` | 🌍 国际聚合 | 面向全球用户的多模型聚合 |
| `account` | 🔑 账号服务 | API Key / 账号代充服务 |
| `free` | 🎗️ 公益站点 | 免费或极低价的社区服务 |

## 🔍 信息核实流程

维护者会在合并 PR 前进行以下核实：

1. 🌐 官网是否可访问
2. 📄 服务条款是否存在
3. 💰 价格信息是否准确
4. 🤖 模型支持是否属实

如果信息无法核实，我们可能会：

- 请求补充证明材料
- 暂缓合并
- 将服务商移至"待确认"分类

---

## 🚫 移除请求

如果你是服务商的负责人，希望将你的服务从列表中移除：

1. 提交 Issue，说明移除原因
2. 使用官方邮箱验证身份
3. 我们会在 48 小时内处理
